# Glossary

## CISO
Chief Information Security Officer.

## KRIs
Key Risk Indicators. Measures that signal risk exposure is rising or staying high.

## KPIs
Key Performance Indicators. Measures that indicate execution performance.

## ZTNA
Zero Trust Network Access. Identity- and context-aware access to applications instead of broad network connectivity.

## CNAPP
Cloud-Native Application Protection Platform. A category covering multiple cloud and workload security capabilities.

## TPRM
Third-Party Risk Management.

## UEBA
User and Entity Behavior Analytics.

## Blast Radius
The practical extent of business, technical, and regulatory impact.
