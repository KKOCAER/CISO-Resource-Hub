# Policy Framework

## Suggested Layers
1. policy
2. standard
3. procedure
4. guideline

## Example
- policy: all privileged access must be strongly authenticated
- standard: phishing-resistant MFA is required for privileged users
- procedure: joiner mover leaver flow for admin accounts
- guideline: preferred admin workstation design

## Why this matters
Without layers, every document becomes either too vague or too rigid.
