# Awesome Threat Intelligence

## Primary and Community Sources
- [CISA Alerts and Advisories](https://www.cisa.gov/news-events/cybersecurity-advisories)
- [MITRE ATT&CK](https://attack.mitre.org/)
- [MISP](https://www.misp-project.org/)
- [OpenCTI](https://www.opencti.io/)
- [SANS Internet Storm Center](https://isc.sans.edu/)
- [FIRST](https://www.first.org/)

## How to use threat intelligence well
Use it to:
- refine likely threat paths
- adjust detection priorities
- tune response readiness
- challenge stale assumptions

Do not use it as a substitute for local telemetry.
