# Awesome Frameworks and Standards

A curated starting point for security leadership and architecture.

## Governance and Program
- [NIST Cybersecurity Framework](https://www.nist.gov/cyberframework)
- [NIST SP 800 Series](https://csrc.nist.gov/publications/sp)
- [CIS Critical Security Controls](https://www.cisecurity.org/controls)
- [ISO/IEC 27001](https://www.iso.org/isoiec-27001-information-security.html)
- [ISO/IEC 27002](https://www.iso.org/standard/75652.html)

## Architecture and Threat
- [MITRE ATT&CK](https://attack.mitre.org/)
- [MITRE ATT&CK Navigator](https://mitre-attack.github.io/attack-navigator/)
- [SABSA](https://sabsa.org/)
- [OWASP ASVS](https://owasp.org/www-project-application-security-verification-standard/)

## Why these matter
These are not interchangeable. Use them for different jobs:
- CSF for program framing
- ATT&CK for adversary behavior mapping
- CIS Controls for practical baseline improvement
- ISO for management system structure
- SABSA for architecture thinking
