# 👔 CISO Liderliği

> Güvenlik programı yönetimi, yönetim kurulu iletişimi ve CISO kariyer gelişimi için kaynaklar.

---

## Kaynaklar

### CISM Sertifikası — ISACA
- **Tür**: Sertifikasyon
- **URL**: https://www.isaca.org/credentialing/cism
- **Açıklama**: Information Security Manager — güvenlik programı yönetimi, yönetişim, risk ve olay yönetimi. 4 domain.

### Gartner CISO Effectiveness Framework
- **Tür**: Araştırma
- **URL**: https://www.gartner.com/en/information-technology/role/chief-information-security-officer
- **Açıklama**: CISO etkinliği için Gartner modeli — yönetim kurulu iletişimi, güvenlik kültürü ve risk dili.

### CISO Mind Map — SANS ⭐ Best Practice
- **Tür**: Rehber
- **URL**: https://www.sans.org/blog/the-ciso-mind-map/
- **Açıklama**: CISO sorumluluk alanlarını görsel olarak haritalayan referans. Program boyutları ve öncelik sıralaması.

## CISO Yönetim Kurulu Raporlama Şablonu

| Boyut | Metrik | Hedef |
|---|---|---|
| Risk | Açık kritik risk sayısı | < 5 |
| Uyum | Kontrol uyumluluk oranı | > %95 |
| Olay | Ortalama tespit süresi (MTTD) | < 24 saat |
| Program | Güvenlik farkındalık katılımı | > %90 |
