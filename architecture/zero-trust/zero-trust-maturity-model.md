# Zero Trust Maturity Model

## Level 1: Fragmented
- MFA inconsistent
- VPN-centric access
- poor application visibility

## Level 2: Controlled
- stronger identity baseline
- segmented high-risk apps
- basic device checks

## Level 3: Scaled
- policy-based access at scale
- privileged access separated
- continuous monitoring aligned

## Level 4: Adaptive
- risk-informed policy
- strong telemetry correlation
- repeatable governance
