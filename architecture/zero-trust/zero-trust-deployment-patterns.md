# Zero Trust Deployment Patterns

## Pattern 1: Remote Workforce First
Start with:
- remote access modernization
- MFA hardening
- application inventory
- device posture checks

## Pattern 2: Privileged Access First
Start with:
- admin identity separation
- admin workstation standards
- conditional access
- PAM and session controls

## Pattern 3: Crown Jewel First
Start with:
- sensitive apps
- payment and transaction systems
- telco core support systems
- customer data platforms

## Advice
Do not attempt a universal cutover. Use targeted waves.
