# Security Strategy Template

## Purpose
Define the target direction of the security program for the next 12 to 36 months.

## Sections
1. business context
2. threat context
3. regulatory context
4. target outcomes
5. priority initiatives
6. operating model shifts
7. investment priorities
8. metrics and checkpoints

## Example Target Outcomes
- reduce identity-related incident exposure
- improve cyber crisis readiness
- reduce control fragmentation
- improve detection confidence
- improve third-party risk visibility

## Guidance
A useful strategy document is short, clear, and tied to business priorities.
