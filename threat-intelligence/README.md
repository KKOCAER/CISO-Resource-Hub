# 🕵️ Tehdit İstihbaratı

> Siber tehditleri proaktif olarak izlemek, analiz etmek ve programa entegre etmek için kaynaklar.

---

## Kaynaklar

### CISA Cybersecurity Advisories
- **Tür**: Kamu Kurumu
- **URL**: https://www.cisa.gov/news-events/cybersecurity-advisories
- **Açıklama**: ABD federal uyarıları — aktif tehdit aktörleri, CVE'ler ve yapılandırma tavsiyeleri.

### MITRE D3FEND
- **Tür**: Bilgi Tabanı
- **URL**: https://d3fend.mitre.org/
- **Açıklama**: ATT&CK saldırı tekniklerine karşılık gelen savunma tekniklerinin ontolojisi.

### ENISA Threat Landscape 2024
- **Tür**: Yıllık Rapor
- **URL**: https://www.enisa.europa.eu/topics/cyber-threats/enisa-threat-landscape
- **Açıklama**: AB siber tehdit manzarası raporu. 10 başlıca tehdit kategorisi ve trend analizi.

### BTK Siber Güvenlik Portalı
- **Tür**: Kamu Kurumu (TR)
- **URL**: https://www.btk.gov.tr/siber-guvenlik
- **Açıklama**: Türkiye BTK siber güvenlik duyuruları ve USOM koordineli uyarılar.

## Tehdit İstihbarat Döngüsü

```
1. YÖNLENDIRME  → Hangi tehditleri izliyoruz?
2. TOPLAMA      → OSINT, dark web, feed'ler
3. İŞLEME       → Normalizasyon, deduplication
4. ANALİZ       → TTP eşleme, önceliklendirme
5. YAYMA        → SIEM, SOC, yönetim raporları
6. GERİ BİLDİRİM→ Program güncelleme
```
